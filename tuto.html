⌨️ Raccourcis clavier
Sur les cartes (champ de bataille)
T
Engager / Désengager la carte sélectionnée tap / untap — équivalent du double-clic
Déplacer les cartes sélectionnées
💡Sélectionne d'abord des cartes (clic simple après 0,5s, Ctrl+clic immédiat, ou rectangle de sélection par glisser), puis utilise les raccourcis.
Ctrl + X
Envoyer les cartes sélectionnées en Exil
Ctrl + G
Envoyer les cartes sélectionnées au Cimetière
Ctrl + D
Remettre les cartes sélectionnées sur le dessus de la Pioche
Ctrl + H
Remettre les cartes sélectionnées dans la Main
Ctrl + C
Activer / désactiver la pastille (badge) sur les cartes sélectionnées
Navigation entre adversaires
Ctrl + ↑
Ouvrir le plateau du premier adversaire (ou de l'adversaire mappé ↑)
Ctrl + ↓
Fermer l'overlay adversaire
Ctrl + ←
Aller à l'adversaire mappé ← (ou adversaire précédent dans l'overlay)
Ctrl + →
Aller à l'adversaire mappé → (ou adversaire suivant dans l'overlay)
🖥️ Sur macOS, remplace Ctrl par ⌘ Cmd pour tous ces raccourcis.

Notes techniques

Serveur WebSocket
Hébergé sur Render (free tier) — peut s'endormir après 15 min d'inactivité, se réveille automatiquement à la prochaine connexion (~30 sec)
Images des cartes
Récupérées depuis Scryfall, version française en priorité, anglaise en fallback
Transfert du deck
Transmis via localStorage entre le deck builder et le plateau de jeu
